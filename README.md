# ReadSffFile

現在開発中...以下下書き

## 基本仕様
MUGENキャラ制作で使用されるSAEのSffファイルを読み取り、スプライトリスト等のデータを取得できます  
SFFv1のみで使用可能です　SFFv2/SFFv2.1は想定していません

## 想定環境
Windows11  
C++言語標準：ISO C++17標準  

上記の環境で動作することを確認しています  
上記以外の環境での動作は保証しません  
  
## 機能一覧
##### 対象のSffファイルを読み込み
　対象ファイルの名前を指定(拡張子は省略可能)  
　ディレクトリパスを指定することで特定のディレクトリのみを検索可能です

##### スプライトリストへのアクセス
　SAEに登録されている番号でその画像の情報にアクセスが可能です

##### 読み込んだSffファイルのBMP出力
　SAEのスプライトリストをそのまま出力するか、重複を除いた画像データのみを出力するか選択が可能です

##### エラー情報の出力切り替え
　ライブラリ側で例外を投げるか否かを指定可能です  
　ログファイルの生成可否も指定可能です
