# ReadSffFile
## 基本仕様
MUGENキャラ制作で使用されるSAEのSffファイルを読み取り、スプライトリスト等のパラメータを取得できます  
SFFv1のみで使用可能です　SFFv2は想定していません

## 想定環境
以下の環境で動作することを確認しています　それ以外の環境は検証していません  
Windows11  
C++言語標準：ISO C++17標準  
  
## 機能一覧
##### 対象のSffファイルを読み込み
　対象ファイルの名前を指定(拡張子は省略可能)  
　ディレクトリパスを指定することで特定のディレクトリのみを検索可能です

##### スプライトリストへのアクセス
　SAEに登録されている番号でその画像の情報にアクセスが可能です

##### 読み込んだSffファイルのBMP出力
　SAEのスプライトリストをそのまま出力するか、重複を除いた画像データのみを出力するか選択が可能です

##### エラー情報の出力切り替え
　ライブラリ側で例外を投げるか否かを指定可能です  
　ログファイルの生成可否も指定可能です
